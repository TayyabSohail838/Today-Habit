# Habit Tracker

A React-based habit tracking app with daily check-ins, streaks, calendar view, and statistics.

## Features

- **Daily tracking** — mark habits complete each day with one click
- **Streaks** — tracks consecutive daily completions per habit
- **Calendar view** — see which days you completed habits
- **Statistics dashboard** — overall completion rates and top habits
- **Data export/import** — backup and restore your data as JSON
- **Local storage** — all data stays in your browser, no server needed

## Getting Started

```bash
npm install
npm run dev
```

Open the URL shown in the terminal (default `http://localhost:5173`).

## Build

```bash
npm run build
```

Output goes to `dist/`.
